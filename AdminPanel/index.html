Booking & Map Use

Bookings can be accepted or denied entering the customer ID in the text field and clicking accept or reject.

To use the map to track bookings enter the customer ID in the text field and click track.

The map can be used to get the route and distance between two points.

Enter the appropiete place name or coordinates into the origin and destination text fields.

Although not fully integrated the text filed below the map can prompt area names.

these names can be used to get a start or end location.


Users & History

The user tab shows user data and history shows user history.

Futhermore users can be searched using their user ID in history.

By entering the user ID in the text field and clicking search.

All data relating to that user will show.


Database

To use the database effectively enter all the data nesscessary before updating an object, with the update button.

Leaving out data will cause data fields to end up blank.

You can also create new objects by entering in new values that the database does not already have and clicking update.

Objects can be removed by entering their ID and clicking the delete button.

To use the search function enter data in one of the text fields and click search.
